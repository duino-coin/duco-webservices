Troubleshooting BETA
My miners are running, but I don't see them

Possible solutions:
1. Make sure you've entered the correct mining key into your miners (it's case sensitive).
2. Once again check that you've entered the correct mining key into your miners.
3. For the third time check that you've entered the correct mining key into your miners. Really.
4. Ensure you've entered the correct username into your miners (it's case sensitive).
5. Check if your miners are actually mining. For AVR and PCs - just read what the mining window says. For ESPs - connect them to a computer and inspect debug messages via serial monitor.
6. Check if our servers aren't having any issues right now - visit status.duinocoin.com.

My miners don't connect to the servers

Possible solutions:
1. Make sure you're not using a proxy/VPN as they're prohibited.
2. Try a different network (cellular or different Wi-Fi).
3. Try a different device and/or a different web browser.
4. Make sure nothing is blocking access to server.duinocoin.com.
5. Restart your internet router.
6. Check if our servers aren't having any issues right now - visit status.duinocoin.com.

My miners get a lot of rejected shares or disconnect randomly

Possible solutions:
1. Check if the configuration is correct. Try a fresh start by deleting existing files.
2. Make sure your internet connection is stable.
3. If using ESPs/AVRs, make sure they get sufficient power.
4. If using ESPs, make sure your router can handle that many clients.
5. If using AVRs, make sure your USB hub/concentrator can handle that many devices (e.g. Raspberry Pis can handle no more than 11 USB devices).
6. If your miners disconnect once every 24 hours then it's all fine - servers restart periodically and so the miners.
